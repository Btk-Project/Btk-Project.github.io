Building prerequisites

Linux

  • G++ with C++17 support
  • SDL2
  • SDL2-ttf
  • SDL2-image
  • fontconfig (optional,for select font)
  • libgif (optional,for display gif)
  • scons or xmake

Windows

  • MINGW or vs2017
  • SDL2
  • SDL2-ttf
  • SDL2-image
  • libgif (optional)
  • scons or xmake

Android

  • Still developing

Build

btk@btk:sudo apt install libsdl2-dev
btk@btk:scons
